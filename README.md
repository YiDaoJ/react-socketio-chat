# react-socketio-chat

A demo project according to Video [Build A Realtime Chat App In ReactJS and NodeJS](https://www.youtube.com/watch?v=NU-HfZY3ATQ&ab_channel=PedroTech) by PedroTech.

start:
1. server
```
cd server
npm install
npm start
```
2. client
```
cd client
npm install
npm start
```
